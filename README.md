# StrikeOnIran – NetLogo Conflict Simulation

A geopolitical simulation model in NetLogo exploring escalation dynamics between the US/Israel and Iran,
inspired by Mohsen.Solhdoost-style scenario analysis.

## Files

- `US,Israel-vs-Iran-v1.4.nlogo` – main NetLogo model
- `PersianGulf.png` - background map
- `env1.png` - The Interface of the Model 
## How to run

1. Open the `.nlogo` file in NetLogo.
2. Click `setup`.
3. Click `go`.
4. Use event buttons (US_Strike, Iranian_Response, etc.) to explore scenarios.
